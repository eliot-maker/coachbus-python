    🚌 BIENVENUE SUR TON INTERFACE COACHBUS 🚌
=====================================================

Salut ! Voici ton pense-bête pour gérer la 
liste des élèves de ton bus directement depuis 
ton terminal (CMD).

-----------------------------------------------------
            LISTE DES COMMANDES DISPONIBLES
-----------------------------------------------------

1. Lancer le site (Local) :
   > python app.py

2. Ajouter un élève un par un :
   > python app.py add

3. Importer toute la liste (écrase l'ancienne) :
   > python app.py import

4. Supprimer UN élève précis (par son nom) :
   > python app.py delete

5. Vider TOUTE la liste d'un coup :
   > python app.py clear

-----------------------------------------------------
Note : Pour l'import, assure-toi que ton fichier 
'liste_eleves.txt' est bien rempli au format NOM;Prénom.

Bonne gestion et bon trajet !
